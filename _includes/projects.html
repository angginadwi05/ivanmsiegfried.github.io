Loan Prediction

  • Logistic Regression, Random-Forest Classification, XGBoost

Proyek ini merupakan proyek yang bertujuan untuk menentukan apakah seseorang dapat diberikan suatu pinjaman atau tidak. Program tersebut diawali dengan proses pengecekan data null dalam dataset, proses mengganti nilai null tersebut dengan nilai baru, pengecekan data duplikat. Lalu, untuk meningkatkan performansi prediksi tersebut, penggunaan teknik label encoding digunakan. Lalu dengan menggunakan berbagai macam model (Logistic Regression, Random Forest Classification, XGBoost), didapatkan XGBoost merupakan model yang paling baik dengan tingkat akurasi hingga 80%.

  • Regression Methods

Program ini merupakan program yang bertujuan untuk menentukan harga rumah dengan beberapa parameter menggunakan regresi linear. Program ini didahului dengan data cleansing. Setelah dibersihkan, data tersebut kemudian dilihat korelasi antara satu parameter dengan parameter lain menggunakan metoda pearson. Setelah ditentukan parameter mana yang akan diramalkan, data tersebut kemudian dilatih menggunakan regresi linear, Lasso, ElasticNet, K-Neighbors-Regressor, Gradient Boosting Regressor. Model-model tersebut kemudian divalidasi untuk melihat keakuratan model dan prediksi yang dihasilkan.

Digit Image Recognition

  • Convolutional Neural Network

Program ini merupakan program pengenalan citra menggunakan convolutional neural network. Citra mendeteksi suatu tepi dengan menggunakan kumpulan-kumpulan metoda matematis semisal konvolusi 2D, Max Pooling yang dikumpulkan dalam suatu sekuensial. Kemudian, bentuk data latih ini dioptimasi menggunakan fungsi Poisson sebagai loss function. Kemudian, sekuensial dan loss function tersebut dikompilasi dan diimplemetasikan kedalam set data yang kita punya. Model tersebut kemudian dievaluasi dengan menghitung tingkat akurasi.
